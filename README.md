# ochan

scaled  websocket  server with python3
a websocket framework use uvicorn 


### install

```

pip install uvicorn
pip install uvitools

```

### run 

```
uvicorn -w 4  server:app    # run  4 workor 

```


### feature

- multi process
- scale well throught redis
- http publish
- support subscribe multi channel
